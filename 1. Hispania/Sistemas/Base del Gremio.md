## Mejoras de la base

- Archivo: mejora para encontrar información sobre lo que esten buscando.
- Barraca: recluta gente para el gremio, pueden ayudar en misiones
- Gaming Hall (casino): cada 7 dias consigues una cantidad de dinero.
> [!warning]- Warning  
> Revisar. Puede romper con el personaje del Torto y como entro en deuda.
- Jardin: dependiendo del tipo que se escoja, se puede ganar dinero vendiendo flores o crear perfumes, conseguir Healer's kits o Potion of Healing, o dos viales de Antitoxin o uno de Basic Poison.
	- Tambien existe Greenhouse, que se parece mucho a Jardin (Garden)
- Laboratorio: puedes pedir que creen Alchemist's Supplies o un Veneno.
- Taverna: consigues una bebida especial con efectos mágicos. Puedes usarla para conseguir información sobre personas que visiten la taverna.
- Santo: permite usar una vez el hechizo Word of Recall hacia este lugar.
- Scriptorium: permite crear copias de libros, panfleto, o un Spell Scroll de Cleric o Wizard (proporcional al nivel).
- Herrero: permite crear objetos que se puedan hacer con Smith's Tools, crear items magicos Common o Uncommon, mejorar armas o armaduras (hablar con DM).
- Teatro: para hacer obras o conciertos.
- Tienda: proporciona materiales o recursos a un precio más economico. Para comprar equipamiento poderoso ya existira una tienda para el Gremio.


## Como conseguir cada mejora

- Dinero
- Subiendo de nivel
- Recompensa por misión
- Conseguir durante la mision
